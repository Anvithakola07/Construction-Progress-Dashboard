# 📊 Interactive Construction Progress Analytics Dashboard

An end-to-end web-based business intelligence solution designed for modern project controls. This interactive dashboard tracks multi-phase construction execution, maps operational baselines, and flags timeline slippage to provide project managers with real-time analytics.

---

## 🔥 Key Features
* **Live Web Deployment:** Hosted entirely via GitHub Pages for easy client or stakeholder access.
* **Progress Performance Tracking:** Interactive progress charts comparing scheduled baselines vs. actual completion rates.
* **Automated Delay Indicators:** Color-coded structural tables classifying tasks instantly into `⚠️ Delayed` or `✅ On Track`.

---

## 🛠️ Tech Stack
* **Frontend UI:** HTML5, CSS3, JavaScript
* **Data Visualization:** Plotly Engine (Embedded Web Layout)
* **Core Analytics & Data Prep:** Python (Pandas)
